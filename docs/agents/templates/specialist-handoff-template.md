# Specialist Handoff Template

Use this when Lead Operations Agent routes a task to a specialist agent.

```text
Subject: [CompeteIQ Ops][ROUTE][P0/P1/P2/P3][Area] [Short event title]

Assigned agent:
[Specialist agent]

Assigned inbox:
[AgentMail inbox]

Secondary agent:
[Optional secondary agent or none]

Issue:
[What happened.]

Known context:
[Safe context only. No secrets, full payment data, or unnecessary personal data.]

Customer impact:
[none / single_customer / multiple_customers / unknown]

Revenue impact:
[none / possible / confirmed / unknown]

Risk level:
[P0 / P1 / P2 / P3]

Approval level:
[Level 0 / Level 1 / Level 2 / Level 3]

Required action:
[Investigate / draft / send routine message / prepare bug report / prepare approval request / summarize]

Allowed action:
[What the specialist can do without further approval.]

Blocked action:
[What the specialist must not do without approval.]

Return to Lead Operations when:
[done / blocked / approval needed / risk level changes / repeated pattern found]

Weekly briefing:
[include critical / include high / include normal / do not include]
```
